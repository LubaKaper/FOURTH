# ECHO — Product Vision

## What it is
A GTM intelligence agent for maternal health software companies. 
Identifies hospitals where public commitments to maternal care 
don't match their postpartum outcomes — and generates personalized 
outbound for the GTM Engineer to act on today.

## The market gap
Tools like Clay, Apollo, and ZoomInfo solve general lead enrichment. 
Nobody has built the maternal health-specific version that speaks 
CMS quality scores, postpartum continuity gaps, and hospital 
commitment language.

## Who pays for this
Companies selling maternal health software to hospitals:
Babyscripts, Maven Clinic, Wildflower Health, Mahmee, 
Bloomlife, Cocoon — and any new entrant in the space.

## Pricing hypothesis
$500-2,000/month per seat. 
10 customers = $60K-$240K ARR.

## Path from class project to real product
- Demo Day: working v1 agent
- MVP validation: 2-3 pilot users from real companies
- Paid pilot: $500/month, 3 companies, 90 days
- Product: v2 features based on pilot feedback

## The moat
The curated, tagged, verified hospital database. 
The agent is replicable. The dataset takes real work 
and that work has real value.
